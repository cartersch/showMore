showMore Jquery Plugin Examples

showMore link reveals more content

This is a section that contains an inner element that is larger than it's parent. The inner element has a couple of different tag styles in it with varying font sizes, padding, etc. The height of this inner div can vary. If the height of the inner div is less than the height of the parent div, then no link will show since all the inner content can be seen. However, when the content of the inner div is too long to show all the content in the parent div, then the plugin will measure the heights of the elements and figure out which one will be displayed.

If there are multiple items in the inner content. The plugin will adjust the height of the last visible element and set it to an overflow of "hidden" while also setting any other elements further down the tree to a display of "none." Clicking the 'Show More' link that is created by the plugin will reveal the entire content of the the inner div.

This plugin has three options:

  • link: This sets the href value of the link. by default it is set to 'javascript:void(0)' so that it goes nowhere and does not add a hashtag to the url
  • styleClass: If you want to apply a specific style to this links you may do so by adding a class name. This class name will go on the span around the link.
  • linkText: If you want to make the link to text read a specific way, you can use this option.

showMore link to external page

This section has a link that takes you to a different page. In this case, the link will take you to the homepage of github.com, but it could do to any page. To change the behavior of a link just use the 'link' option in the plugin.
